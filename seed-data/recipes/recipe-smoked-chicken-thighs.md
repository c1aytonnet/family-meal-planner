ingredients:
  - { name: Chicken thighs, amount: 8 bone-in, category: Meat }
  - { name: Baby potatoes, amount: 2 lb, category: Produce }
  - { name: Olive oil, amount: 2 tbsp, category: Pantry }
  - { name: Smoked paprika, amount: 2 tsp, category: Pantry }
  - { name: Garlic powder, amount: 1 tsp, category: Pantry }
  - { name: Lemon, amount: 1, category: Produce }
  - { name: Parsley, amount: 1 bunch, category: Produce }
steps:
  - Toss potatoes with oil, salt, and half the spices. Roast until crisp-tender.
  - Season chicken thighs and smoke at 275F until they reach 175F.
  - Toss potatoes with lemon zest and parsley just before serving.
  - Rest the chicken for 10 minutes and serve with pan juices.
notes:
  - Great for Sunday dinner with enough leftovers for lunch bowls.
  - Use boneless thighs if you need a shorter cook.

Smoky, family-friendly chicken with a potato side that feels special without getting complicated.
